# Hands-On 6: Selenium Automation using Pytest

## Objective
Implement automated testing using the Pytest framework with Selenium WebDriver.

## Technologies Used
- Python
- Selenium
- Pytest
- pytest-html

## Tasks Performed
- Installed and configured Pytest
- Created reusable browser fixture using conftest.py
- Implemented automated test cases
- Executed test suite using Pytest
- Generated HTML test reports

## Learning Outcome
- Understood Pytest framework
- Created automated Selenium test cases
- Used fixtures for browser management
- Generated professional HTML reports

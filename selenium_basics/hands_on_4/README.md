# Hands-On 4: Browser Automation using Selenium

## Objective
Implement browser automation using Selenium WebDriver by performing basic browser operations and web interactions.

## Technologies Used
- Python 3.12+
- Selenium WebDriver
- ChromeDriver
- webdriver-manager

## Tasks Performed
- Launched Chrome browser
- Navigated between web pages
- Performed Google search automation
- Located web elements using ID and Class Name
- Automated login functionality
- Captured browser screenshots
- Handled multiple browser tabs/windows
- Executed browser in headless mode

## Learning Outcome
- Understood Selenium WebDriver setup
- Automated browser actions
- Worked with browser navigation
- Performed basic web element interactions
- Captured screenshots
- Managed multiple tabs/windows
- Executed headless browser automation

# Hands-On 9: Department Statistics API

## Objective
Create a Department Statistics API to retrieve department-wise information.

## Technologies Used
- Python 3.12.6
- Django 6.0.7
- Django REST Framework
- Django ORM
- SQLite

## Steps Performed
1. Created a custom Department Statistics API.
2. Retrieved department details using the department ID.
3. Calculated the total number of courses.
4. Calculated the total number of students.
5. Calculated the total number of enrollments.
6. Returned the results as a JSON response.

## Output
- Department Statistics API created successfully.
- Department-wise statistics returned in JSON format.
- API tested successfully.

## Learning Outcome
- Created a custom REST API.
- Used Django ORM `filter()` and `count()` methods.
- Retrieved department-wise analytical data.

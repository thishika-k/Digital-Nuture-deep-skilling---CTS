# Hands-On 5: JWT Authentication using Django REST Framework

## Objective
Secure REST APIs using JSON Web Token (JWT) authentication with Django REST Framework Simple JWT.

## Technologies Used
- Python 3.12.6
- Django 6.0.7
- Django REST Framework
- Django REST Framework Simple JWT
- SQLite

## Steps Performed
1. Installed `djangorestframework-simplejwt`.
2. Configured JWT authentication in `settings.py`.
3. Added JWT token generation and refresh endpoints.
4. Protected API views using `IsAuthenticated`.
5. Created a superuser.
6. Generated JWT access and refresh tokens.
7. Accessed protected APIs using the Bearer token.

## Output
- JWT package installed successfully.
- Token generation endpoint configured.
- Access and refresh tokens generated.
- Protected APIs accessed successfully using JWT authentication.

## Learning Outcome
- Learned JWT authentication using Django REST Framework.
- Configured access and refresh tokens.
- Secured REST APIs with `IsAuthenticated`.
- Successfully authenticated and accessed protected endpoints.

# Hands-On 8: Dashboard API

## Objective
Create a Dashboard API to provide an overview of the Course Management System.

## Technologies Used
- Python 3.12.6
- Django 6.0.7
- Django REST Framework
- SQLite

## Steps Performed
1. Created a custom Dashboard API view.
2. Used Django ORM `count()` function.
3. Returned the total number of departments, courses, students, and enrollments.
4. Registered the Dashboard API endpoint.
5. Tested the API successfully.

## Output
- Dashboard API created successfully.
- Aggregated data returned as JSON.
- API tested successfully.

## Learning Outcome
- Created a custom REST API endpoint.
- Used Django ORM aggregation with `count()`.
- Returned summarized data in JSON format.

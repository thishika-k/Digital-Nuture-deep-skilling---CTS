
# Hands-On 4: Implementing CRUD Operations using Django REST Framework

## Objective
Implement CRUD (Create, Read, Update, Delete) operations for the Course Management System using Django REST Framework Generic Views.

## Technologies Used
- Python 3.12.6
- Django 6.0.7
- Django REST Framework
- SQLite

## Steps Performed
1. Created List/Create API views.
2. Implemented Retrieve/Update/Delete API views.
3. Configured URL routing.
4. Tested CRUD endpoints using the browser or an API client.

## Output
- Created course records.
- Retrieved course details.
- Updated existing records.
- Deleted records successfully.

## Learning Outcome
- Learned CRUD implementation using Generic Views.
- Understood REST API endpoint design.
- Tested API operations successfully.

# Hands-On 6: Filtering, Searching and Ordering in Django REST Framework

## Objective
Implement filtering, searching, and ordering functionalities for REST APIs using Django REST Framework.

## Technologies Used
- Python 3.12.6
- Django 6.0.7
- Django REST Framework
- django-filter
- SQLite

## Steps Performed
1. Installed `django-filter`.
2. Added `django_filters` to `INSTALLED_APPS`.
3. Configured `DjangoFilterBackend`, `SearchFilter`, and `OrderingFilter`.
4. Enabled filtering by department.
5. Enabled searching by course name and code.
6. Enabled ordering by credits and course name.
7. Tested all API endpoints successfully.

## Output
- Filtering implemented.
- Searching implemented.
- Ordering implemented.
- API responses verified.

## Learning Outcome
- Learned filtering using `django-filter`.
- Implemented search functionality.
- Implemented ordering of API results.
- Improved API usability.

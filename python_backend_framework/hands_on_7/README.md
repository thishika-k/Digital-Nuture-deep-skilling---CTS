# Hands-On 7: API Documentation using Swagger

## Objective
Generate interactive API documentation using Swagger UI and ReDoc.

## Technologies Used
- Python 3.12.6
- Django 6.0.7
- Django REST Framework
- drf-yasg

## Steps Performed
1. Installed `drf-yasg`.
2. Configured Swagger in `backend/urls.py`.
3. Generated Swagger UI documentation.
4. Generated ReDoc documentation.
5. Verified all REST API endpoints.

## Output
- Swagger UI generated successfully.
- ReDoc generated successfully.
- Interactive API documentation available.

## Learning Outcome
- Learned Swagger API documentation.
- Generated interactive API documentation.
- Tested APIs directly from Swagger UI.

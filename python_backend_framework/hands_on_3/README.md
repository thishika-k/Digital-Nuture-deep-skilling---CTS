# Hands-On 3: Building REST APIs using Django REST Framework

## Objective
Develop REST APIs using Django REST Framework by creating serializers, API views, and URL routing for the Course Management System.

## Software Requirements
- Python 3.13.7
- Django 6.0.6
- Django REST Framework

## Steps Performed
1. Installed Django REST Framework.
2. Registered `rest_framework` in `INSTALLED_APPS`.
3. Created `serializers.py`.
4. Implemented serializers for Department, Course, Student, and Enrollment.
5. Prepared the project for REST API development.

## Output
- Django REST Framework installed successfully.
- REST Framework registered successfully.
- Serializers created.

## Learning Outcome
- Learned how to install and configure Django REST Framework.
- Understood the purpose of serializers.
- Prepared Django models for REST API development.
